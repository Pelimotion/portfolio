How to use

1. Select a client or category from the sidebar.
2. Edit the Description, Release text, and Deliverables.
3. Click Save & Export JSON to download the updated content.json.
4. Replace the content.json in the portfolio folder.
5. Run python3 deploy_system.py to publish changes live.
