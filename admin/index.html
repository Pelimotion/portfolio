📖 How to use

1. Select a section from the sidebar:

  • 🏠 Landing Page — Brand name, navigation buttons, footer text
  • 🎬 Portfolio — Hero text, contact info, social links, footer quotes
  • 📄 Curriculum — Name, title, profile, stats, experience, education, skills, contact
  • ✉️ Cover Letter — Greeting, body paragraphs, target role/company, sign-off
  • Clients — Description, release text, and deliverable tags for each project
  • Categories — Description and deliverables for project sub-categories

2. Edit the fields. Changes auto-save when you switch sections.

3. Click Publish to Vercel to instantly push changes directly to the live site.

Tip: Use ⌘S / Ctrl+S to quick-save at any time.

🗂️ What each field controls
Client Description Shown below the client name in the portfolio gallery header
Release Text Editorial text shown in the video player modal
Deliverables Tag pills shown below description (e.g. "Social Media Campaign")
Hero Name The massive title in the portfolio hero section
Footer Quotes The two statements at the bottom of the portfolio page
CV Stats The 4 metric boxes in the curriculum hero (e.g. "08+ Years")
